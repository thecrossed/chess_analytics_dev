FAQ

Common questions about data fetch and PGN analysis.

How many days can I fetch game data?

The selected date range cannot exceed 120 days.

Can I upload PGN by file and by paste?

Yes. You can upload a PGN file or paste PGN text. If they conflict, keep one source.

Can I export analysis results to CSV?

Yes. Both game stats and PGN analysis outputs can be downloaded as CSV.

Which platforms are supported?

Game data fetching supports Chess.com and Lichess.

How are eval score and best move obtained in PGN analysis?

Eval score comes from engine evaluation after the played move. Best move and best move eval are engine recommendations from the position before that move.

What is Stockfish?

Stockfish is a strong open-source chess engine used to evaluate positions and suggest best moves.

Which extra PGN analysis fields are available now?

PGN analysis can now include eval gap, accuracy, book move status, opening ECO, opening name, and white or black clock values when the PGN contains clock annotations.

Can I customize which PGN analysis columns are visible?

Yes. The PGN analysis table keeps key move columns pinned and lets you toggle Engine, Opening, and Clock column groups, or customize individual columns.
