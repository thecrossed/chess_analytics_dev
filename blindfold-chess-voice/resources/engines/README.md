# Stockfish engine binaries

Place one unmodified Stockfish executable in this directory for local development:

- `stockfish`
- `stockfish.exe`
- `stockfish-macos-arm64`
- `stockfish-macos-x86_64`
- `stockfish-linux-x86_64`
- `stockfish-windows-x86_64.exe`

Release builds should include the correct Stockfish binary automatically and complete the license/source metadata under `licenses/`.
